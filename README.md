# Exam Invigilation Scheduling System

## Overview
An automated exam invigilation scheduling system developed using Python and Genetic Algorithms.

## Features
- Faculty allocation based on availability
- Hall-wise invigilator assignment
- Conflict-free scheduling
- Balanced workload distribution
- Student and hall management

## Technologies Used
- Python
- Pandas
- Genetic Algorithm (PyGAD)

## Files
- faculty.csv
- students.csv
- availability.csv
- exam_hall.csv
- exam_inv_hall.py
